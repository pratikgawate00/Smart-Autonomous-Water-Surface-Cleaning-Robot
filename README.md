# Smart-Autonomous-Water-Surface-Cleaning-Robot

## 📌 Project Overview
The Smart Autonomous Water Surface Cleaning Robot is an IoT-based system designed to remove floating waste from water bodies such as lakes, rivers, and ponds. The robot operates autonomously using sensors and a propulsion mechanism while being powered by solar energy.

---

## 🎯 Objectives
- To clean water surfaces efficiently
- To reduce human effort in waste collection
- To promote eco-friendly and sustainable cleaning using solar energy
- To implement automation in environmental conservation

---

## ⚙️ Features
- 🤖 Autonomous navigation
- 🌞 Solar-powered system
- 📷 Camera-based monitoring
- 🧺 Debris collection net
- 🚤 Propeller-based movement
- ♻️ Eco-friendly operation
- 
🛠️ Components Used
| Sr No | Component               | Specification                                               | Purpose                                    |
| ----- | ----------------------- | ----------------------------------------------------------- | ------------------------------------------ |
| 1     | Solar Panel             | 12V, 20W–50W, high-efficiency (monocrystalline)             | Supplies renewable power to the robot      |
| 2     | Camera Module           | 1080p / 4K resolution, wide-angle (120°–160°), low power    | Used for monitoring and navigation         |
| 3     | Debris Collection Net   | Nylon/plastic mesh, medium mesh size, high storage capacity | Collects floating waste from water surface |

| 6     | Battery                 | 12V rechargeable battery (Li-ion / Lead-acid)               | Stores energy for continuous operation     |
| 7     | Motor Driver            | L298N / BTS7960, dual channel, high current support         | Controls motor direction and speed         |
| 8     | Ultrasonic Sensor       | Range: 2cm – 400cm, high accuracy                           | Obstacle detection                         |
| 9     | Float Body (Hull)       | Lightweight, waterproof material (PVC / Fiber)              | Keeps robot afloat                         |
| 10    | Wi-Fi Module (optional) | ESP8266 / ESP32 built-in Wi-Fi                              | Enables remote monitoring (IoT)            |


## 🧠 Working Principle
1. The robot floats on the water surface.
2. Solar panels provide power to the system.
3. The propeller drives the robot forward.
4. The debris collection net collects floating waste.
5. A camera monitors surroundings for navigation and obstacle detection.

---

## 📐 Design Overview
- Length: ~1.5m – 1.8m
- Width: ~0.8m
- Solar panel mounted on top
- Front opening for waste collection
- Rear propeller for movement
